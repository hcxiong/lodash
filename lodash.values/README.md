# lodash.values v3.0.0

The [modern build](https://github.com/lodash/lodash/wiki/Build-Differences) of the [Lo-Dash](https://lodash.com/) function `_.values` exported as a [Node.js](http://nodejs.org/)/[io.js](https://iojs.org/) module.

## Dive in

See the [documentation](https://lodash.com/docs#values) or [package source](https://github.com/lodash/lodash/blob/3.0.0-npm-packages/lodash.values/index.js) for more details.

## Installation

Using npm:

```bash
$ {sudo -H} npm i -g npm
$ npm i --save lodash.values

$ {sudo -H} npm i -g lodash.values
$ npm ln lodash.values
```

In Node.js/io.js:

```js
var values = require('lodash.values');
```
